# SkyWalking Ingester

## 快速启动

```sh
go build .

export PROJECT=<YOUR_PROJECT>
export ENDPOINT=<YOUR_ENDPOINT>
export ACCESS_KEY=<YOUR_ACCESS_KEY>
export SECURITY_KEY=<YOUR_ACCESS_SECURITY_KEY>
export TRACE_INSTANCE=<YOUR_TRACE_INSTANCE>
export NAMESPACE=<YOUR_TOPIC_NAMESPACE>
export BOOTSTRAP_SERVERS=<YOUR_BOOTSTRAP_SERVERS>
export GROUP=<YOUR_CONSUMER_GROUP_ID>

./skywalking-ingester
```
